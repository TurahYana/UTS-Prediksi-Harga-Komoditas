# Prediksi Harga Komoditas Pertanian dengan Machine Learning

Repository ini berisi proyek Machine Learning berbasis **Multiple Linear Regression** untuk memprediksi harga keseimbangan (*Modal Price*) komoditas pertanian di pasar. 
Proyek ini mencakup keseluruhan *pipeline* mulai dari pembersihan data, *feature engineering* (waktu dan musiman), *encoding* kategorikal, hingga evaluasi model statistik.


## Persyaratan Sistem
Pastikan Anda menggunakan **Python 3.8+**.
